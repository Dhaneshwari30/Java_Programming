# Java Fundamentals

This repository contains beginner Java programs created while learning Java fundamentals.

## Topics Covered

- JVM, JDK, JRE
- Variables
- Data Types
- Type Casting
- Operators
- User Input using Scanner
- User Input using BufferedReader
- Basic Calculations
- Unit Conversions

## Programs

### Basic Calculations
- Addition
- Subtraction
- Multiplication
- Division
- Remainder
- Square
- Cube
- Average of Three Numbers
- Swap Two Numbers
- Percentage Calculator

### Conversions
- Celsius to Fahrenheit
- Fahrenheit to Celsius
- Kilometer to Meter
- Meter to Kilometer

### User Input
- Scanner Input Demo
- BufferedReader Input Demo
- Character Input
- Student Information System
